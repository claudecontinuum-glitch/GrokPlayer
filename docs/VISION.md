# Visión GrokPlayer

GrokPlayer no compite con servicios de streaming. Es un **framework de experiencia** para archivos musicales locales:

- **Local-first** — tus archivos, tu máquina.
- **Visible** — la música se *ve* (Starship synthwave v1).
- **Compartible** — sesiones JSON y API para que agentes y humanos co-curaten el momento, no la piratería.

El objetivo es que alguien quiera **sentarse a escuchar** por el placer de la interfaz y el ritual compartido.
